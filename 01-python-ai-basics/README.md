# 01 - Python AI Basics

This directory contains foundational Python practice artifacts completed as part of the Generative AI and LLM application development coursework.

The purpose of this section is to demonstrate basic programming readiness required for AI-native software development workflows, including data handling, functions, modular programming, and simple automation patterns.

## Artifacts

| File | Purpose |
|---|---|
| `python_foundations.py` | Demonstrates variables, functions, control flow, list/dictionary handling, and basic text processing. |
| `data_handling_exercise.py` | Demonstrates simple structured data processing relevant to AI/LLM experiment logs. |
| `ai_prompt_dataset_utils.py` | Provides small utilities for cleaning and preparing prompt-response examples. |
| `exercise-notes.md` | Notebook-style exercise notes with explanation and expected learning outcomes. |
| `learning-reflection.md` | Reflection connecting Python basics to AI Native SDLC research. |

## How to Run

```bash
python python_foundations.py
python data_handling_exercise.py
python ai_prompt_dataset_utils.py
```
