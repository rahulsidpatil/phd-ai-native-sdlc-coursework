# Python AI Basics - Exercise Notes

## Objective

The objective of this module is to demonstrate basic Python programming skills required for AI and LLM application development.

## Exercises Completed

### 1. Python Foundations

File: `python_foundations.py`

Concepts practiced:

- Variables and strings
- Lists
- Dictionaries
- Functions
- Loops
- Basic text normalization
- Keyword counting

Learning outcome:

This exercise demonstrates how basic Python constructs can be used to prepare and inspect text, which is a common activity in prompt engineering and LLM application development.

### 2. Structured Data Handling

File: `data_handling_exercise.py`

Concepts practiced:

- Dataclasses
- Lists of structured records
- Filtering
- Aggregation
- Summary reporting

Learning outcome:

This exercise demonstrates how Python can be used to organize learning records or experiment logs. Similar techniques are useful when analyzing prompt experiments, model responses, or SDLC automation results.

### 3. Prompt Dataset Utilities

File: `ai_prompt_dataset_utils.py`

Concepts practiced:

- Text cleaning
- Record validation
- Category-based filtering
- Modular utility functions

Learning outcome:

This exercise demonstrates simple preprocessing steps for prompt-response examples, which are relevant to LLM evaluation and AI-assisted SDLC experimentation.

## Relevance to AI Native SDLC

Python is commonly used for prototyping AI-powered workflows. Foundational Python skills are important for building experiments related to prompt engineering, LLM APIs, RAG applications, test generation, documentation automation, and developer productivity tools.
