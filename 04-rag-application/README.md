# 04 - RAG Application

This directory contains Retrieval-Augmented Generation learning artifacts.

The artifacts describe a simple RAG workflow and include lightweight prototype code that demonstrates document chunking, keyword-based retrieval, and answer construction without external dependencies.

## Artifacts

| File | Purpose |
|---|---|
| `rag-concept-notes.md` | Conceptual overview of RAG and its SDLC relevance. |
| `simple_rag_prototype.py` | Lightweight local RAG-style prototype using keyword retrieval. |
| `sample_documents.md` | Small sample knowledge base for local experimentation. |
| `rag-evaluation-checklist.md` | Checklist for evaluating RAG outputs. |
| `learning-reflection.md` | Reflection connecting RAG to AI Native SDLC. |
