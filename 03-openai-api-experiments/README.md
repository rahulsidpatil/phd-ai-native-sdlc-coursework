# 03 - LLM API Experiments

This directory contains LLM API integration practice artifacts.

The files are written as educational templates and mock examples. They avoid including any API keys or secrets.

## Artifacts

| File | Purpose |
|---|---|
| `api_call_template.py` | Safe template showing how an LLM API integration can be structured. |
| `mock_llm_client.py` | Mock LLM client for local experimentation without external API calls. |
| `prompt_response_logger.py` | Utility for logging prompt-response experiments. |
| `experiment-plan.md` | Plan for conducting LLM API experiments. |
| `learning-reflection.md` | Reflection connecting LLM APIs to AI Native SDLC. |

## Security Note

Never commit API keys, tokens, `.env` files, or secrets to GitHub.
