# Learning Reflection - LLM API Experiments

## Summary

This module helped me understand how LLM capabilities can be integrated into software workflows through APIs.

## Key Learnings

1. API integration enables AI functionality inside real applications.
2. Prompt design and response handling are both important.
3. Logging is useful for evaluating prompt-response experiments.
4. Secrets must be handled securely using environment variables or secret managers.
5. Mock clients are useful before connecting to real services.

## Connection to AI Native SDLC

LLM APIs can support AI-native SDLC tooling by enabling automated or assisted workflows for requirements, coding, testing, documentation, and maintenance. API integration is therefore an important practical foundation for my research area.
