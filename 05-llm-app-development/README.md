# 05 - LLM App Development

This directory contains artifacts related to building LLM-powered applications.

The focus is on application structure, use case design, workflow decomposition, and a small local prototype for an AI-assisted SDLC helper.

## Artifacts

| File | Purpose |
|---|---|
| `app-architecture.md` | Describes a sample LLM app architecture for AI Native SDLC. |
| `sdlc_assistant_prototype.py` | Lightweight command-line prototype for SDLC assistance. |
| `use-cases.md` | Documents possible LLM app use cases for AI Native SDLC. |
| `risk-and-controls.md` | Notes risks and controls for LLM app development. |
| `learning-reflection.md` | Reflection connecting LLM app development to PhD research. |

## Academic Integrity Note

These artifacts are original practice and documentation files prepared for academic review.
