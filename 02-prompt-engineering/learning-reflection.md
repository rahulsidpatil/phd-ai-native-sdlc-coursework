# Learning Reflection - Prompt Engineering

## Summary

This module helped me understand how prompt design affects the quality, structure, and usefulness of LLM-generated outputs.

## Key Learnings

1. A vague prompt usually produces generic output.
2. Role, context, constraints, and output format improve response quality.
3. Prompt engineering is useful for SDLC artifact generation and review.
4. Human validation is essential because LLM outputs may contain assumptions or errors.

## Connection to AI Native SDLC

Prompt engineering is directly relevant to AI Native SDLC because AI tools can assist in requirements analysis, user story generation, test case design, code explanation, documentation, and review workflows.
