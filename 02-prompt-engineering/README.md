# 02 - Prompt Engineering

This directory contains prompt engineering experiments and documentation artifacts prepared as part of the Generative AI and LLM application development coursework.

The artifacts focus on prompt clarity, role-based prompting, structured outputs, evaluation prompts, and prompt patterns relevant to AI Native SDLC.

## Artifacts

| File | Purpose |
|---|---|
| `prompt-patterns.md` | Documents reusable prompt patterns for SDLC-related use cases. |
| `prompt-experiments.md` | Contains example prompt experiments and observations. |
| `requirements-to-user-stories-prompts.md` | Demonstrates prompts for converting requirements into user stories. |
| `test-case-generation-prompts.md` | Demonstrates prompts for generating test cases from requirements. |
| `learning-reflection.md` | Reflection connecting prompt engineering to AI Native SDLC research. |

## Academic Integrity Note

These artifacts contain original prompt templates, observations, and reflections. They do not reproduce proprietary Udemy course material.
